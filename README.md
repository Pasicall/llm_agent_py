# LLM Agent 

基于 asyncio 和 MCP 协议的高性能 LLM Agent 系统，支持多工具协同、RAG 检索增强生成与异步任务调度。

## 核心特性

### 1. Agent 任务循环调度框架
- **异步任务调度**：基于 asyncio 实现完整的任务循环调度机制
- **流式响应解析**：实时解析 LLM 流式输出，支持工具调用结果上下文管理
- **多工具协同**：支持动态工具注册与自动上下文注入

### 2. MCP 协议客户端
- **标准化实现**：完整实现 Model Context Protocol 客户端
- **工具集成**：成功集成 2 个工具 Server（fetch、filesystem），包含 15 个原生工具
- **生命周期管理**：支持工具的动态注册、调用与资源清理

### 3. 向量数据库
- **从零实现**：基于纯 Python 实现向量数据库，无需依赖第三方向量库
- **余弦相似度算法**：实现高效的语义检索算法
- **性能表现**：
  - 10 文档：1.25ms
  - 100 文档：12.21ms
  - 1000 文档：117.98ms
  - 性能线性可扩展

### 4. RAG 检索增强生成
- **Embedding 流水线**：集成 BAAI/bge-large-zh-v1.5 预训练模型
- **语义检索**：基于向量相似度的 Top-K 检索
- **召回准确率**：Hit Rate@5 达到 100%


## 核心模块说明

### Agent.py

实现完整的 Agent 任务循环调度框架：
- 工具调用路由与结果管理
- 流式响应实时解析
- 异步任务调度

### MCPClient.py

MCP 协议客户端实现：
- 与 MCP Server 的连接管理
- 工具列表获取与调用
- 生命周期管理

### VectorStore.py

向量数据库核心实现：
- 向量存储与索引
- 余弦相似度计算
- Top-K 检索算法

### EmbeddingRetriver.py

Embedding 检索器：
- 文档嵌入
- 查询嵌入
- 语义检索
